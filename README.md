# Vite React batteries included

A template mono repo that set up many useful libraries and is ready to use

## Root app

- pnpm
- prettier
- husky + lint-stage

## Vite-app

- Vite
- React
- Typescript
- Tailwind CSS
- React hook form + dev tool
- React-query

[Clone](https://github.com/nhducit/vite-react-batteries-included/generate)
